Introduzione a AngularJS

AngularJS è un framework JavaScript open source sviluppato da Google che estende l'HTML con nuovi attributi e rende le pagine web dinamiche senza riscrivere la struttura. È la versione 1.x del framework Angular, ancora ampiamente usata in progetti legacy.

CaratteristicaDescrizione
MVCSepara Model, View e Controller
Data BindingSincronizzazione automatica tra modello e vista
DirettiveEstende l'HTML con nuovi attributi e tag
Dependency InjectionGestione automatica delle dipendenze
TestabileProgettato per il testing unitario
⚠️ Attenzione: AngularJS (v1.x) ha raggiunto la fine del supporto ufficiale nel dicembre 2021. Per nuovi progetti si raccomanda Angular moderno (v2+). AngularJS rimane importante per manutenere applicazioni esistenti.

Setup: includere AngularJS

HTML — primo esempio AngularJS
<!DOCTYPE html>
<html ng-app>
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body>

<p>Nome: <input type="text" ng-model="nome"></p>
<p>Ciao, {{ nome }}!</p>

</body>
</html>

L'attributo ng-app dice ad AngularJS di controllare quella porzione di pagina. Le doppie graffe {{ }} visualizzano valori del modello. ng-model crea il binding bidirezionale: modifica il campo e il testo si aggiorna istantaneamente.

Espressioni

Le espressioni AngularJS sono scritte tra doppie graffe {{ espressione }}. Possono contenere operatori matematici, stringhe, oggetti e array — ma non strutture di controllo come if o loop.

HTML — espressioni
<!DOCTYPE html>
<html ng-app>
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body>

<!-- Operazioni matematiche -->
<p>5 + 3 = {{ 5 + 3 }}</p>
<p>10 * 4 = {{ 10 * 4 }}</p>

<!-- Stringhe -->
<p>{{ "Ciao" + " " + "Mondo!" }}</p>

<!-- Oggetto -->
<p ng-init="persona = {nome:'Marco', eta:30}">
  {{ persona.nome }} ha {{ persona.eta }} anni.
</p>

<!-- Array -->
<p ng-init="corsi=['HTML','CSS','JavaScript']">
  Terzo corso: {{ corsi[2] }}
</p>

</body>
</html>
📋 ng-init inizializza variabili direttamente nel template. È comodo per esempi rapidi, ma nella pratica i dati vanno definiti nel Controller.

Moduli

Un modulo AngularJS è il contenitore dell'applicazione. Si crea con angular.module() e si collega alla pagina tramite ng-app="nomeModulo".

HTML + JS — creare un modulo
<!DOCTYPE html>
<html ng-app="miaApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body>

<div ng-controller="MioController">
  <h2>{{ messaggio }}</h2>
  <p>Versione: {{ versione }}</p>
</div>

<script>
// Crea il modulo
var app = angular.module('miaApp', []);

// Aggiunge un controller al modulo
app.controller('MioController', function($scope) {
  $scope.messaggio = 'Benvenuto in AngularJS!';
  $scope.versione  = '1.8.3';
});
</script>

</body>
</html>

Il secondo parametro di angular.module() è un array di dipendenze. Un array vuoto [] significa che il modulo non dipende da altri moduli.

Direttive

Le direttive sono attributi HTML speciali che iniziano con ng-. Estendono l'HTML aggiungendo comportamenti dinamici. AngularJS ne include decine, e puoi crearne di personalizzate.

DirettivaFunzione
ng-appInizializza l'applicazione AngularJS
ng-modelBinding bidirezionale con un campo input
ng-controllerCollega un controller alla vista
ng-repeatRipete un elemento HTML per ogni item di una lista
ng-ifMostra/nasconde un elemento in base a una condizione
ng-show / ng-hideMostra/nasconde (con display) in base a una condizione
ng-clickEsegue codice al click
ng-classAggiunge classi CSS dinamicamente
ng-styleApplica stili inline dinamicamente
ng-initInizializza variabili di scope
HTML — direttive ng-repeat e ng-if
<!DOCTYPE html>
<html ng-app="miaApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
</head>
<body>
<div ng-controller="ListaCtrl">

  <h3>I miei corsi:</h3>
  <ul>
    <li ng-repeat="corso in corsi">
      {{ corso.nome }}
      <span ng-if="corso.completato"> ✅</span>
      <span ng-if="!corso.completato"> 🔄</span>
    </li>
  </ul>

  <p ng-show="mostraTotale">
    Totale corsi: {{ corsi.length }}
  </p>
  <button ng-click="mostraTotale = !mostraTotale">Mostra/Nascondi totale</button>

</div>
<script>
angular.module('miaApp', []).controller('ListaCtrl', function($scope) {
  $scope.corsi = [
    { nome: 'HTML', completato: true },
    { nome: 'CSS',  completato: true },
    { nome: 'AngularJS', completato: false }
  ];
  $scope.mostraTotale = false;
});
</script>
</body>
</html>

ng-model

ng-model crea il data binding bidirezionale tra un campo di input e una variabile dello scope. Ogni modifica al campo aggiorna istantaneamente il modello, e viceversa.

HTML — ng-model e binding in tempo reale
<!DOCTYPE html>
<html ng-app="miaApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: Arial, sans-serif; padding: 20px; }

    .anteprima { background: #f0f4ff; padding: 16px; border-radius: 8px; margin-top: 12px; }
  </style>
</head>
<body>
<div ng-controller="ModelloCtrl">

  <label>Nome:</label>
  <input type="text" ng-model="nome" placeholder="Inserisci nome">

  <label>Cognome:</label>
  <input type="text" ng-model="cognome" placeholder="Inserisci cognome">

  <label>Età:</label>
  <input type="number" ng-model="eta" placeholder="Età">

  <div class="anteprima">
    <strong>Anteprima in tempo reale:</strong><br>
    {{ nome }} {{ cognome }}, {{ eta }} anni
  </div>

</div>
<script>
angular.module('miaApp', []).controller('ModelloCtrl', function($scope) {
  $scope.nome    = 'Marco';
  $scope.cognome = 'Rossi';
  $scope.eta     = 30;
});
</script>
</body>
</html>

Data Binding

Il data binding è il meccanismo che sincronizza automaticamente il modello (dati JavaScript) con la vista (HTML). AngularJS usa il binding bidirezionale: ogni cambiamento in uno si riflette nell'altro senza codice aggiuntivo.

HTML — calcolatrice con data binding
<!DOCTYPE html>
<html ng-app>
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: Arial, sans-serif; padding: 20px; }
    input { width: 80px; padding: 8px; font-size: 1.1rem; text-align: center; }
    .risultato { font-size: 1.4rem; font-weight: bold; color: #4f46e5; margin-top: 16px; }
  </style>
</head>
<body>

<h2>Calcolatrice live</h2>
<input type="number" ng-model="a" ng-init="a=10">
 + 
<input type="number" ng-model="b" ng-init="b=5">

<div class="risultato">
  Risultato: {{ a * 1 + b * 1 }}
</div>

<p>Somma: {{ a * 1 + b * 1 }}</p>
<p>Differenza: {{ a - b }}</p>
<p>Prodotto: {{ a * b }}</p>
<p>Divisione: {{ (b != 0) ? (a / b | number:2) : 'Divisione per zero!' }}</p>

</body>
</html>

Controller

Il controller è una funzione JavaScript che gestisce i dati e la logica della vista. Usa $scope per esporre variabili e funzioni all'HTML.

HTML — controller con metodi
<!DOCTYPE html>
<html ng-app="negozio">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: Arial, sans-serif; padding: 20px; }
    button { padding: 8px 16px; margin: 4px; cursor: pointer; border: none; border-radius: 4px; }
    .aggiungi { background: #4f46e5; color: #fff; }
    .rimuovi  { background: #ef4444; color: #fff; }
    .carrello { background: #f0fdf4; padding: 16px; border-radius: 8px; margin-top: 12px; }
  </style>
</head>
<body>
<div ng-controller="NegozioCtrl">

  <h2>🛒 Carrello ({{ carrello.length }} prodotti)</h2>

  <div ng-repeat="prodotto in prodotti">
    {{ prodotto.nome }} — €{{ prodotto.prezzo }}
    <button class="aggiungi" ng-click="aggiungi(prodotto)">+ Aggiungi</button>
  </div>

  <div class="carrello">
    <strong>Nel carrello:</strong>
    <p ng-repeat="item in carrello">{{ item.nome }} — €{{ item.prezzo }}</p>
    <p ng-if="carrello.length === 0">Carrello vuoto</p>
    <strong>Totale: €{{ totale() }}</strong>
    <br><button class="rimuovi" ng-click="svuota()">Svuota carrello</button>
  </div>

</div>
<script>
angular.module('negozio', []).controller('NegozioCtrl', function($scope) {
  $scope.prodotti = [
    { nome: 'Laptop', prezzo: 899 },
    { nome: 'Mouse',  prezzo: 25  },
    { nome: 'Tastiera', prezzo: 55 }
  ];
  $scope.carrello = [];
  $scope.aggiungi = function(p) { $scope.carrello.push(p); };
  $scope.svuota   = function()  { $scope.carrello = [];    };
  $scope.totale   = function()  {
    return $scope.carrello.reduce(function(sum, p) { return sum + p.prezzo; }, 0);
  };
});
</script>
</body>
</html>

Scope

Lo $scope è l'oggetto condiviso tra controller e vista. È il "ponte" che rende disponibili dati e funzioni all'HTML. Esiste uno scope radice ($rootScope) accessibile da tutta l'applicazione, e scope locali per ogni controller.

HTML — scope padre e figlio
<!DOCTYPE html>
<html ng-app="scopeApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: Arial; padding: 16px; }
    .padre { border: 2px solid #4f46e5; padding: 16px; border-radius: 8px; }
    .figlio { border: 2px solid #0d9488; padding: 12px; border-radius: 8px; margin-top: 12px; }
  </style>
</head>
<body>
<div ng-controller="PadreCtrl" class="padre">
  <strong>Scope Padre</strong>
  <p>Nome padre: {{ nomePadre }}</p>

  <div ng-controller="FiglioCtrl" class="figlio">
    <strong>Scope Figlio</strong>
    <p>Eredita dal padre: {{ nomePadre }}</p>
    <p>Proprio del figlio: {{ nomeFiglio }}</p>
  </div>
</div>

<script>
angular.module('scopeApp', [])
  .controller('PadreCtrl', function($scope) {
    $scope.nomePadre = 'Mario';
  })
  .controller('FiglioCtrl', function($scope) {
    $scope.nomeFiglio = 'Luigi';
  });
</script>
</body>
</html>
💡 Lo scope figlio eredita le proprietà dello scope padre (prototypal inheritance di JavaScript). Se lo scope figlio modifica una proprietà ereditata, crea una copia locale senza modificare il padre.

Filtri

I filtri trasformano i dati prima che vengano mostrati nella vista. Si applicano con il carattere pipe | nelle espressioni o nel codice JS.

FiltroEffettoEsempio
uppercaseTutto maiuscolo{{ testo | uppercase }}
lowercaseTutto minuscolo{{ testo | lowercase }}
currencyFormato valuta{{ prezzo | currency:"€" }}
numberDecimali fissi{{ num | number:2 }}
dateFormato data{{ data | date:"dd/MM/yyyy" }}
orderByOrdina arrayng-repeat="x in arr | orderBy:'nome'"
filterFiltra arrayng-repeat="x in arr | filter:cerca"
limitToLimita elementing-repeat="x in arr | limitTo:3"
HTML — ricerca e filtro in tempo reale
<!DOCTYPE html>
<html ng-app="filtroApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: Arial; padding: 16px; }
    input { padding: 8px; width: 100%; border: 1px solid #ccc; border-radius: 4px; margin-bottom: 12px; }
    th { cursor: pointer; background: #4f46e5; color: #fff; padding: 8px; }
    td { padding: 6px 8px; border-bottom: 1px solid #eee; }
  </style>
</head>
<body>
<div ng-controller="FiltroCtrl">

  <input type="text" ng-model="cerca" placeholder="🔍 Cerca prodotto...">

  <table>
    <tr>
      <th ng-click="ordine='nome'">Nome ↕</th>
      <th ng-click="ordine='prezzo'">Prezzo ↕</th>
    </tr>
    <tr ng-repeat="p in prodotti | filter:cerca | orderBy:ordine">
      <td>{{ p.nome }}</td>
      <td>{{ p.prezzo | currency:"€" }}</td>
    </tr>
  </table>

</div>

angular.module('filtroApp', []).controller('FiltroCtrl', function($scope) {
  $scope.ordine = 'nome';
  $scope.prodotti = [
    { nome: 'Laptop',    prezzo: 899 },
    { nome: 'Tablet',    prezzo: 350 },
    { nome: 'Smartphone',prezzo: 599 },
    { nome: 'Monitor',   prezzo: 299 },
    { nome: 'Tastiera',  prezzo: 55  }
  ];
});
</script>
</body>
</html>

Servizi

I servizi AngularJS sono oggetti singleton riutilizzabili tra i controller. AngularJS include servizi predefiniti (come $http, $timeout, $interval) e permette di crearne di personalizzati.

HTML — $timeout e $interval
<!DOCTYPE html>
<html ng-app="servizioApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: Arial; padding: 20px; }
    .orologio { font-size: 2.5rem; font-weight: bold; color: #4f46e5; }
    button { padding: 8px 16px; margin: 6px 4px; cursor: pointer; border-radius: 4px; border: none; background: #4f46e5; color: #fff; }
  </style>
</head>
<body>
<div ng-controller="ServizioCtrl">

  <h2>⏱ Orologio live</h2>
  <div class="orologio">{{ ora }}</div>

  <button ng-click="avvia()">▶ Avvia</button>
  <button ng-click="ferma()">⏹ Ferma</button>

  <p>Contatore clic sul Avvia: {{ clic }}</p>

</div>
<script>
angular.module('servizioApp', [])
  .controller('ServizioCtrl', function($scope, $interval, $filter) {
    var timer;
    $scope.clic = 0;

    function aggiorna() {
      $scope.ora = $filter('date')(new Date(), 'HH:mm:ss');
    }
    aggiorna();

    $scope.avvia = function() {
      if (!timer) {
        $scope.clic++;
        timer = $interval(aggiorna, 1000);
      }
    };
    $scope.ferma = function() {
      if (timer) { $interval.cancel(timer); timer = null; }
    };
  });
</script>
</body>
</html>

Servizio personalizzato con factory

JS — factory personalizzata
<!DOCTYPE html>
<html ng-app="factoryApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style> body { font-family: Arial; padding: 20px; } </style>
</head>
<body>
<div ng-controller="CalcoloCtrl">
  <h2>IVA Calculator</h2>
  Prezzo netto: <input type="number" ng-model="netto" ng-init="netto=100">
  <p>IVA (22%): €{{ iva.calcola(netto) | number:2 }}</p>
  <p>Totale: €{{ iva.totale(netto) | number:2 }}</p>
</div>

<script>
var app = angular.module('factoryApp', []);

// Factory personalizzata
app.factory('iva', function() {
  return {
    calcola: function(n) { return n * 0.22; },
    totale:  function(n) { return n * 1.22; }
  };
});

// Controller che usa la factory
app.controller('CalcoloCtrl', function($scope, iva) {
  $scope.iva = iva;
  $scope.netto = 100;
});
</script>
</body>
</html>

$http

Il servizio $http esegue richieste HTTP (AJAX) verso API esterne o file JSON locali. Restituisce una promise con i metodi .then() e .catch().

HTML — $http GET da API pubblica
<!DOCTYPE html>
<html ng-app="httpApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <style>
    body { font-family: Arial; padding: 20px; }
    .post { border: 1px solid #e2e8f0; padding: 12px; border-radius: 8px; margin: 8px 0; }
    .post h4 { margin: 0 0 6px; color: #4f46e5; text-transform: capitalize; }
    .caricamento { color: #64748b; font-style: italic; }
    button { padding: 8px 16px; background: #4f46e5; color: #fff; border: none; border-radius: 4px; cursor: pointer; }
  </style>
</head>
<body>
<div ng-controller="HttpCtrl">

  <h2>Post da JSONPlaceholder API</h2>
  <button ng-click="carica()">📥 Carica post</button>

  <p class="caricamento" ng-if="caricamento">Caricamento in corso...</p>
  <p ng-if="errore" style="color:red">Errore: {{ errore }}</p>

  <div class="post" ng-repeat="p in posts | limitTo:5">
    <h4>{{ p.title }}</h4>
    <p>{{ p.body }}</p>
  </div>

</div>
<script>
angular.module('httpApp', []).controller('HttpCtrl', function($scope, $http) {
  $scope.posts      = [];
  $scope.caricamento = false;
  $scope.errore     = null;

  $scope.carica = function() {
    $scope.caricamento = true;
    $scope.errore = null;
    $http.get('https://jsonplaceholder.typicode.com/posts')
      .then(function(risposta) {
        $scope.posts = risposta.data;
      })
      .catch(function(err) {
        $scope.errore = 'Impossibile caricare i dati.';
      })
      .finally(function() {
        $scope.caricamento = false;
      });
  };
});
</script>
</body>
</html>

DOM

AngularJS gestisce il DOM tramite direttive. Le principali per il controllo della visualizzazione sono ng-show, ng-hide, ng-class, ng-style e ng-disabled.

HTML — ng-class e ng-style dinamici
<!DOCTYPE html>

Animazioni

Il modulo ngAnimate aggiunge animazioni CSS automaticamente quando le direttive AngularJS mostrano o nascondono elementi. Basta includere il modulo e definire le classi CSS.

HTML — animazioni con ngAnimate
<!DOCTYPE html>
<html ng-app="animApp">
<head>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular.min.js"></script>
  <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.8.3/angular-animate.min.js"></script>
  <style>
    body { font-family: Arial; padding: 20px; }
    .box { padding: 20px; background: #4f46e5; color: #fff; border-radius: 8px;
           margin: 12px 0; font-size: 1.1rem; font-weight: bold; text-align: center; }

    /* Animazione ng-show/ng-hide */
    .box.ng-hide-add    { animation: fadeOut 0.4s; }
    .box.ng-hide-remove { animation: fadeIn  0.4s; }

    /* Animazione ng-repeat */
    .item.ng-enter { animation: slideIn 0.4s; }
    .item.ng-leave { animation: slideOut 0.3s; }

    @keyframes fadeIn   { from { opacity:0; transform:scale(.9); } to { opacity:1; transform:scale(1); } }
    @keyframes fadeOut  { from { opacity:1; } to { opacity:0; } }
    @keyframes slideIn  { from { opacity:0; transform:translateX(-20px); } to { opacity:1; transform:none; } }
    @keyframes slideOut { from { opacity:1; } to { opacity:0; transform:translateX(20px); } }

    button { padding: 8px 16px; margin: 4px; border: none; border-radius: 4px; cursor: pointer;
             background: #4f46e5; color: #fff; }
    .item  { padding: 8px 12px; border: 1px solid #e2e8f0; border-radius: 6px; margin: 4px 0; }
  </style>
</head>
<body>
<div ng-controller="AnimCtrl">

  <button ng-click="visibile = !visibile">Toggle banner</button>
  <div class="box" ng-show="visibile">🎉 Animazione con ngAnimate!</div>

  <hr>
  <button ng-click="aggiungi()">+ Aggiungi</button>
  <button ng-click="rimuovi()">- Rimuovi</button>
  <div class="item" ng-repeat="i in items">Elemento {{ i }}</div>

</div>
<script>
angular.module('animApp', ['ngAnimate']).controller('AnimCtrl', function($scope) {
  $scope.visibile = true;
  $scope.items = [1, 2, 3];
  var cnt = 4;
  $scope.aggiungi = function() { $scope.items.push(cnt++); };
  $scope.rimuovi  = function() { if ($scope.items.length) $scope.items.pop(); };
});
</script>
</body>
</html>

    { nome: 'Alice Bianchi',  email: 'alice@esempio.it',  tel: '333-111-2222' },
    { nome: 'Marco Rossi',    email: 'marco@esempio.it',  tel: '347-555-6666' },
    { nome: 'Sofia Ferrari',  email: 'sofia@esempio.it',  tel: '' }
  ];
  $scope.aggiungi = function(form) {
    if (form.$valid) {
      $scope.contatti.push(angular.copy($scope.nuovo));
      $scope.nuovo = {};
      form.$setPristine();
    }
  };

    $scope.contatti.splice(i, 1);
