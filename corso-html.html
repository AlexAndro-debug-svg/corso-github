1. Introduzione a HTML

HTML (HyperText Markup Language) è il linguaggio con cui sono costruite tutte le pagine web. Non è un linguaggio di programmazione: è un linguaggio di marcatura, cioè serve a “etichettare” le parti di un documento per dire al browser cosa è un titolo, cosa un paragrafo, cosa un'immagine.

Ogni pagina che vedi su internet — compresa questa — è scritta in HTML. Imparare l'HTML è il primo passo per diventare sviluppatori web.

💡 In breve: l'HTML descrive la struttura di una pagina (titoli, testo, immagini, link), il CSS si occupa dell'aspetto (colori, posizioni) e JavaScript del comportamento (interattività).

3. Elementi e tag

        
Un elemento HTML è composto quasi sempre da un tag di apertura, un contenuto e un tag di chiusura:

        
<p> contenuto </p>

        
Il tag di chiusura ha la barra /. Alcuni elementi non hanno contenuto e si chiudono da soli, come <br> (a capo) o <hr> (linea orizzontale).
