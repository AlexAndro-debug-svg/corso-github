Corso di CSS completo in italiano

Benvenuto nel corso di CSS più completo in italiano. CSS (Cascading Style Sheets, fogli di stile a cascata) è il linguaggio che definisce l'aspetto delle pagine web: colori, font, spaziature, posizionamento, animazioni e layout responsive.

In questo corso trovi tutti gli argomenti del CSS spiegati passo passo, dalle basi fino alle tecniche avanzate come Flexbox, le pseudo-classi e le funzioni matematiche. Ogni capitolo ha esempi reali col codice colorato e un editor “Provalo tu” per scrivere e vedere subito il risultato nel browser.

💡 HTML, CSS e JavaScript: l'HTML è la struttura, il CSS è l'aspetto e JavaScript è il comportamento. Padroneggiare il CSS è indispensabile per chiunque voglia creare siti web professionali.
Esempio

Introduzione al CSS

Il CSS nasce nel 1996 per separare il contenuto (HTML) dalla presentazione (stile). Prima del CSS, l'aspetto veniva definito direttamente nell'HTML, rendendo le pagine difficili da mantenere.

Con il CSS puoi controllare colori, dimensioni, font, spaziature, posizioni e animazioni di tutti gli elementi, e modificare l'intero sito cambiando un solo file. È uno standard del W3C supportato da tutti i browser.

  • Manutenzione facile: un foglio di stile per tutto il sito.
  • Pagine più leggere e veloci.
  • Design responsive: stili diversi per telefono, tablet e desktop.

La sintassi del CSS

Una regola CSS è composta da un selettore e da un blocco di dichiarazioni racchiuse tra parentesi graffe. Ogni dichiarazione è una coppia proprietà: valore; e termina con il punto e virgola.

Esempio: h1 { color: blue; font-size: 32px; } applica a tutti gli <h1> il colore blu e la dimensione 32px.

Esempio

I selettori CSS

I selettori indicano a quali elementi applicare lo stile. I principali:

  • Selettore di elemento: p (tutti i paragrafi)
  • Selettore di classe: .nome (elementi con class="nome")
  • Selettore di id: #nome (l'unico elemento con quell'id)
  • Selettore universale: * (tutti gli elementi)
  • Selettore di gruppo: h1, h2, p (più selettori insieme)
Esempio

Come inserire il CSS

Esistono tre modi per applicare il CSS a una pagina:

  • CSS inline: con l'attributo style nel singolo tag (da usare poco).
  • CSS interno: dentro <style> nell'<head>, vale per quella pagina.
  • CSS esterno: in un file .css collegato con <link> — è il metodo migliore, perché vale per tutto il sito.

Se più stili sono in conflitto vince quello più specifico e, a parità, l'ultimo dichiarato (la “cascata”).

Esempio

I commenti CSS

I commenti spiegano il codice e vengono ignorati dal browser. In CSS si scrivono tra /* e */ e possono occupare più righe. Sono utili per organizzare fogli di stile lunghi.

Esempio

Errori comuni nel CSS

Conoscere gli errori più frequenti fa risparmiare ore di debug:

  • Punto e virgola mancante dopo una dichiarazione.
  • Parentesi graffa non chiusa.
  • Nome di proprietà o valore errato (es. colour invece di color).
  • Unità di misura dimenticata (es. width: 100 invece di 100px).
  • Selettore sbagliato (punto per le classi, cancelletto per gli id).
⭐ Usa gli strumenti per sviluppatori del browser (tasto F12) per individuare le regole CSS non applicate.

I colori in CSS

I colori si indicano in diversi formati: per nome (red), in esadecimale (#ff0000), in RGB (rgb(255,0,0)), in RGBA con trasparenza, o in HSL (tonalità, saturazione, luminosità).

Si applicano con color (testo) e background-color (sfondo).

Esempio

Gli sfondi (background)

Le proprietà dello sfondo: background-color (colore), background-image (immagine), background-repeat (ripetizione), background-position (posizione), background-size (dimensione, es. cover). Si possono riassumere nella scorciatoia background.

Si possono usare anche gradienti con linear-gradient() o radial-gradient().

Esempio

I bordi (border)

Il bordo si definisce con border indicando spessore, stile e colore: border: 2px solid #333;. Stili comuni: solid, dashed, dotted. Con border-radius si arrotondano gli angoli; con valori alti si ottengono cerchi.

Esempio

I margini (margin)

Il margine è lo spazio esterno all'elemento, che lo distanzia dagli altri. Si imposta con margin (tutti i lati) o per singolo lato (margin-top, -right, -bottom, -left). Con margin: 0 auto; si centra orizzontalmente un blocco di larghezza definita.

Esempio

Il padding

Il padding è lo spazio interno tra il bordo e il contenuto. Si imposta come il margine: padding: 20px; o per singolo lato. Più padding rende un elemento più “arioso” e leggibile.

💡 Margin vs Padding: il margin spinge fuori (verso gli altri), il padding spinge dentro (allarga la scatola attorno al contenuto).
Esempio

Altezza e larghezza

Le proprietà width e height impostano dimensioni. Si possono usare pixel (px), percentuali (%) o unità relative alla finestra (vw, vh). Con min-width/max-width e min-height/max-height si impostano limiti.

Esempio

Il box model

Ogni elemento HTML è una scatola formata da quattro strati, dall'interno verso l'esterno: content (contenuto), padding (spazio interno), border (bordo) e margin (spazio esterno). Capire il box model è fondamentale per controllare gli spazi.

Con box-sizing: border-box; la larghezza include padding e bordo, rendendo i calcoli più semplici (è una pratica molto diffusa).

Esempio

Il contorno (outline)

L'outline è una linea tracciata fuori dal bordo, che non occupa spazio e non sposta gli altri elementi. È molto usato per evidenziare l'elemento attivo (focus), importante per l'accessibilità da tastiera.

Esempio

Formattare il testo

Proprietà principali del testo: color, text-align (allineamento), text-decoration (sottolineatura), text-transform (maiuscole/minuscole), line-height (interlinea), letter-spacing (spaziatura tra lettere), text-indent (rientro).

Esempio

I font

Con font-family scegli il carattere (sempre con alternative di riserva), con font-size la dimensione, con font-weight lo spessore (es. bold), con font-style il corsivo. Puoi importare font esterni come Google Fonts.

Esempio

Le icone

Le icone si aggiungono con librerie come Font Awesome o Google Material Icons (richiamate via <link>), oppure usando direttamente le emoji Unicode, che non richiedono librerie esterne e si stilizzano come testo.

Esempio

Lo stile dei link

I link possono cambiare aspetto nei loro stati: :link (non visitato), :visited (visitato), :hover (mouse sopra), :active (mentre si clicca). Spesso si tolgono la sottolineatura con text-decoration: none e si aggiunge un effetto al passaggio del mouse.

Esempio

Le liste

Con il CSS controlli l'aspetto delle liste: list-style-type cambia il segno (disco, cerchio, quadrato, numeri), list-style: none li rimuove (utile per i menu di navigazione), list-style-position ne imposta la posizione.

Esempio

Le tabelle

Le tabelle si rendono leggibili con border-collapse: collapse (unisce i bordi), padding nelle celle, colori alternati per le righe (:nth-child(even)) e uno sfondo per l'intestazione.

Esempio

La proprietà display

display determina come un elemento viene reso. Valori principali: block (occupa tutta la riga), inline (resta in riga), inline-block (in riga ma con dimensioni), none (nasconde l'elemento), flex e grid (per i layout).

Esempio

Max-width

max-width imposta la larghezza massima di un elemento: può restringersi su schermi piccoli ma non supera il valore indicato. Combinato con margin: 0 auto è il modo classico per creare un contenitore centrato e responsive.

Esempio

Il posizionamento (position)

La proprietà position controlla come un elemento viene posizionato:

  • static: predefinito, nel flusso normale.
  • relative: spostato rispetto alla sua posizione originale.
  • absolute: posizionato rispetto al primo antenato posizionato.
  • fixed: fisso rispetto alla finestra (resta visibile allo scroll).
  • sticky: si “incolla” quando raggiunge un bordo durante lo scroll.
Esempio

Gli offset di posizione

Insieme a position si usano le proprietà di offset top, right, bottom e left per spostare l'elemento dai rispettivi bordi. Con position: relative spostano rispetto alla posizione originale; con absolute/fixed rispetto al riferimento.

Esempio

Z-index (sovrapposizione)

Quando gli elementi si sovrappongono, z-index decide quale sta davanti: valori più alti stanno sopra. Funziona solo su elementi posizionati (relative, absolute, fixed, sticky).

Esempio

Overflow (contenuto in eccesso)

overflow gestisce il contenuto che esce da un elemento di dimensioni fisse: visible (predefinito, esce), hidden (taglia), scroll (sempre barre di scorrimento), auto (barre solo se servono).

Esempio

Float (testo attorno)

float fa “galleggiare” un elemento a sinistra o a destra, con il testo che gli scorre attorno (storicamente usato per i layout). Con clear si impedisce ad altri elementi di affiancarsi.

💡 Per i layout moderni oggi si preferiscono Flexbox e Grid; il float resta utile per far scorrere il testo attorno a un'immagine.
Esempio

Inline-block

display: inline-block unisce i vantaggi di inline (gli elementi stanno in riga) e block (si possono impostare larghezza, altezza, padding e margini). Utile per creare pulsanti o caselle affiancate.

Esempio

Allineare gli elementi

Per centrare: il testo con text-align: center; un blocco con margin: 0 auto; in verticale e orizzontale con Flexbox usando justify-content e align-items.

Esempio

I combinatori

I combinatori selezionano elementi in base alla loro relazione:

  • Discendente div p: tutti i p dentro un div.
  • Figlio div > p: solo i figli diretti.
  • Fratello adiacente h2 + p: il p subito dopo un h2.
  • Fratelli generici h2 ~ p: tutti i p fratelli successivi.
Esempio

Le pseudo-classi

Le pseudo-classi applicano stili in base allo stato o alla posizione di un elemento, usando i due punti: :hover (mouse sopra), :focus (in focus), :first-child, :last-child, :nth-child(n) (ennesimo figlio), :checked (per checkbox/radio).

Esempio

Gli pseudo-elementi

Gli pseudo-elementi (con doppio due punti) stilizzano una parte specifica: ::first-line (prima riga), ::first-letter (prima lettera), ::before e ::after (inseriscono contenuto prima/dopo), ::selection (testo selezionato).

Esempio

Opacità e trasparenza

opacity imposta la trasparenza da 0 (invisibile) a 1 (opaco). In alternativa, per rendere trasparente solo lo sfondo (e non il testo) si usa un colore rgba().

Esempio

Le barre di navigazione

Una barra di navigazione si costruisce da una lista (<ul>) togliendo i puntini e mettendo le voci in orizzontale con Flexbox o inline-block. È il menu tipico di ogni sito.

Esempio

I menu a tendina

Un menu a tendina mostra altre voci al passaggio del mouse, usando un contenitore con position: relative e un sottomenu position: absolute nascosto con display: none e mostrato con :hover.

Esempio

Galleria di immagini

Una galleria si crea disponendo più immagini in griglia con Flexbox o Grid, aggiungendo bordo, padding e un effetto al passaggio del mouse. È perfetta per portfolio e cataloghi.

Esempio

Gli image sprites

Uno sprite è un'unica immagine che contiene più icone. Con background-position si mostra solo la porzione desiderata. Vantaggio: una sola richiesta al server invece di tante, quindi pagine più veloci.

💡 Tecnica storica molto usata per le icone; oggi spesso sostituita da font di icone o SVG, ma ancora utile da conoscere.

Selettori di attributo

Si possono selezionare elementi in base ai loro attributi: [type="text"] (valore esatto), [href^="https"] (inizia con), [href$=".pdf"] (finisce con), [class*="col"] (contiene). Molto utili per i form e i link.

Esempio

Lo stile dei form

Con il CSS si rendono eleganti caselle, aree di testo e pulsanti: padding, bordi arrotondati, colore al focus. Form curati aumentano l'usabilità e la fiducia degli utenti.

Esempio

I contatori

I contatori numerano automaticamente gli elementi con counter-reset (inizializza), counter-increment (incrementa) e content: counter(nome). Utili per numerare sezioni o capitoli in modo dinamico.

Esempio

Le unità di misura

Le unità si dividono in assolute (px) e relative:

  • px: pixel, misura fissa.
  • %: percentuale rispetto al genitore.
  • em: relativo al font dell'elemento.
  • rem: relativo al font radice (html).
  • vw/vh: percentuale della larghezza/altezza della finestra.
⭐ Usa rem per i font e le spaziature: rende il sito più coerente e accessibile.
Esempio

L'ereditarietà

Alcune proprietà (come color, font-family, font-size) si ereditano: impostandole sul genitore, valgono anche per i figli. Altre (come border o margin) no. Con il valore inherit puoi forzare l'ereditarietà.

Esempio

La specificità

Quando più regole colpiscono lo stesso elemento, vince quella con specificità maggiore. In ordine crescente: selettore di elemento < classe < id < stile inline. A parità di specificità vince l'ultima regola scritta.

Esempio

La regola !important

!important forza una dichiarazione a prevalere su tutte le altre, ignorando la specificità. È potente ma va usato con parsimonia: ne abusa rende il CSS difficile da gestire e da correggere.

Esempio

Le funzioni matematiche

Il CSS include funzioni matematiche: calc() esegue calcoli mescolando unità diverse (es. calc(100% - 40px)); min() e max() scelgono il valore minimo/massimo; clamp() imposta un valore fluido tra un minimo e un massimo (ideale per font responsive).

Esempio

Ottimizzazione del CSS

Un CSS ottimizzato rende il sito più veloce e facile da mantenere:

  • Usa un file esterno unico e minificato (senza spazi e commenti) in produzione.
  • Sfrutta le variabili CSS (--colore) per non ripetere i valori.
  • Evita selettori troppo complessi e regole duplicate.
  • Raggruppa le regole simili e riusa le classi.
Esempio

L'accessibilità

Un sito accessibile è usabile da tutti, comprese le persone con disabilità. Buone pratiche CSS:

  • Garantisci un contrasto sufficiente tra testo e sfondo.
  • Non rimuovere l'outline dal focus senza un'alternativa visibile.
  • Usa unità relative (rem) per rispettare le impostazioni dell'utente.
  • Non comunicare informazioni solo con il colore.
♿ L'accessibilità è una competenza fondamentale e sempre più richiesta: rende il web migliore per tutti.

Il layout di un sito web

Un sito tipico ha: header (testata), nav (menu), main con eventuale aside (barra laterale) e footer. Oggi questi blocchi si dispongono con Flexbox e CSS Grid, che gestiscono righe e colonne in modo semplice e responsive.
