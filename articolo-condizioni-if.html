Le condizioni if, elif ed else in Python

📅 6 giugno 2026 — di Alessandro Simone

Un programma diventa davvero utile quando sa prendere decisioni: "se l'utente è maggiorenne fai questo, altrimenti fai quest'altro". In Python questo si fa con le strutture condizionali if, elif ed else. In questa guida le vediamo passo passo, con esempi che puoi provare subito.

L'istruzione if

La parola if (in inglese "se") esegue un blocco di codice solo se una condizione è vera:

eta = 20
if eta >= 18:
    print("Sei maggiorenne")

Se la condizione eta >= 18 è vera, viene stampato il messaggio. Se è falsa, Python salta semplicemente il blocco. Come per i cicli, l'indentazione (gli spazi a inizio riga) indica quali istruzioni dipendono dall'if.

L'istruzione else

Con else (in inglese "altrimenti") diciamo a Python cosa fare quando la condizione è falsa:

eta = 15
if eta >= 18:
    print("Sei maggiorenne")
else:
    print("Sei minorenne")

Ora il programma copre entrambi i casi: o si esegue il blocco dell'if, o quello dell'else. Mai tutti e due insieme.

L'istruzione elif

E se i casi sono più di due? Entra in gioco elif (contrazione di "else if"), che permette di controllare più condizioni in sequenza:

voto = 7
if voto >= 8:
    print("Ottimo!")
elif voto >= 6:
    print("Sufficiente")
else:
    print("Insufficiente")

Python controlla le condizioni dall'alto verso il basso e si ferma alla prima vera. Con voto = 7, salta il primo if, entra nell'elif e stampa "Sufficiente". Puoi mettere quanti elif vuoi.

Gli operatori di confronto

Le condizioni si costruiscono con gli operatori di confronto, che restituiscono sempre True o False:

  • == uguale a (attenzione: due segni uguale!)
  • != diverso da
  • > e < maggiore e minore
  • >= e <= maggiore o uguale, minore o uguale

L'errore più frequente è usare un solo = (che serve ad assegnare un valore) al posto di == (che serve a confrontare). Ricordalo bene!

Combinare le condizioni: and, or, not

Possiamo unire più condizioni con gli operatori logici:

eta = 20
patente = True
if eta >= 18 and patente:
    print("Puoi guidare")
  • and: vero solo se entrambe le condizioni sono vere;
  • or: vero se almeno una è vera;
  • not: inverte il valore (vero diventa falso e viceversa).

Un esempio completo

numero = int(input("Inserisci un numero: "))
if numero > 0:
    print("Il numero è positivo")
elif numero < 0:
    print("Il numero è negativo")
else:
    print("Il numero è zero")

Questo piccolo programma classifica un numero in positivo, negativo o zero: un classico esercizio da prima lezione che usa tutte e tre le strutture.

In sintesi

Con if esegui codice se una condizione è vera, con else gestisci il caso contrario e con elif controlli più alternative. Ricorda di usare == per confrontare e di curare l'indentazione. Le condizioni, insieme ai cicli, sono il cuore di ogni programma: padroneggiarle ti apre le porte a qualsiasi progetto.
