Introduzione a MYSQL

MySQL è uno dei database relazionali più diffusi al mondo. Usa il linguaggio SQL per gestire i dati.

💡 In questo corso puoi eseguire query reali direttamente nel browser! Clicca su Provalo tu ▶. È già pronto un database di esempio con le tabelle studenti (id, nome, voto, classe, classe_id) e classi (id, aula, classe). Il motore è SQLite: la sintassi SQL è quasi identica a MySQL.
Esempio

Leggere i dati: SELECT

SELECT seleziona le colonne da una tabella. L'asterisco * indica tutte le colonne.

Esempio

Filtrare: WHERE

WHERE filtra le righe in base a una condizione.

Esempio

Ordinare: ORDER BY

ORDER BY ordina i risultati. ASC crescente, DESC decrescente.

Esempio

Inserire: INSERT

INSERT INTO aggiunge una nuova riga alla tabella.

Esempio

Modificare: UPDATE

UPDATE modifica righe esistenti. Usa sempre WHERE, altrimenti modifichi tutto!

Esempio

Cancellare: DELETE

DELETE elimina righe. Anche qui il WHERE è fondamentale.

Esempio

Funzioni di aggregazione

COUNT, AVG, MAX, MIN, SUM calcolano valori su più righe.

Esempio

Raggruppare: GROUP BY

GROUP BY raggruppa le righe per calcolare valori per gruppo.

Esempio

Unire tabelle: JOIN

Il JOIN combina righe di due tabelle collegate da una colonna comune.

Esempio

Creare una tabella

CREATE TABLE definisce una nuova tabella con le sue colonne e i tipi di dato.
