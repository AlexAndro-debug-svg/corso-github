E ora?

Cos'è Go

Go (chiamato anche Golang) è un linguaggio di programmazione moderno, creato da Google nel 2009. È compilato, veloce come il C ma semplice da leggere come Python, ed è pensato per costruire software efficiente e scalabile.

Perché imparare Go

  • ⚡ Veloce: compila in codice macchina nativo
  • 🧹 Semplice: poche parole chiave, sintassi pulita e leggibile
  • 🔀 Concorrenza nativa: le goroutine rendono facile fare più cose insieme
  • ☁️ Cloud e backend: è il linguaggio di Docker, Kubernetes e moltissimi servizi web
  • 💼 Richiestissimo: tra i linguaggi meglio pagati sul mercato

Introduzione a Go

Go è nato per risolvere problemi reali di Google: compilazione lenta, codice difficile da mantenere e gestione complicata della concorrenza. Il risultato è un linguaggio minimale ma potente.

Caratteristiche principali

  • Tipizzazione statica: gli errori di tipo si scoprono già in compilazione
  • Garbage collector: gestisce la memoria automaticamente
  • Libreria standard ricca: web server, JSON, crittografia inclusi
  • Un solo eseguibile: il programma compilato non ha dipendenze esterne

Dove si usa Go

Backend di servizi web e API, strumenti per il cloud (Docker, Kubernetes, Terraform sono scritti in Go), microservizi, strumenti da riga di comando e sistemi ad alte prestazioni.

💡 Go è un ottimo secondo linguaggio: se conosci già le basi (magari da Python o C), lo impari in pochissimo tempo.

Iniziare con Go

Per programmare in Go sul tuo computer scarica il compilatore da go.dev/dl. In alternativa, per esercitarti subito senza installare nulla, usa il Go Playground online.

La sintassi di Go

Il codice Go è organizzato in modo rigoroso e pulito. Ecco gli elementi che ritroverai in ogni programma.

  • package — dichiara a quale pacchetto appartiene il file
  • import — importa altri pacchetti
  • func — definisce una funzione
  • Le parentesi graffe { } delimitano i blocchi
Anatomia del codice Go
📘 In Go il punto e virgola a fine riga non si scrive: lo aggiunge il compilatore. La parentesi graffa di apertura { deve stare sulla stessa riga di func.

I commenti

I commenti sono note ignorate dal compilatore, utili a spiegare il codice. Go ha due forme, identiche a C, Java e JavaScript.

Commenti su una riga e su più righe

Le variabili

Una variabile è un contenitore con un nome per un valore. In Go si dichiara con var oppure, dentro una funzione, con la forma breve :=.

Dichiarare variabili
📘 La forma breve := funziona solo dentro le funzioni e deduce il tipo dal valore. A livello globale usa sempre var.
⚠️ Go è severo: una variabile dichiarata ma mai usata provoca un errore di compilazione. Niente codice morto!

Le costanti

Una costante è un valore che non può cambiare dopo essere stato definito. Si dichiara con const.

Definire costanti
💡 Usa le costanti per valori fissi (aliquote, limiti, messaggi): rendono il codice più chiaro e impediscono modifiche accidentali.

Le tre funzioni di stampa

Gli array

Un array è una collezione di elementi dello stesso tipo con dimensione fissa, decisa alla creazione.

Creare e usare array
⚠️ La dimensione di un array fa parte del suo tipo e non può cambiare. Per collezioni che crescono si usano le slice (prossima lezione).

Le slice

Una slice è come un array ma di dimensione dinamica: può crescere e rimpicciolirsi. È la struttura più usata in Go.

Slice e append
💡 Regola pratica: usa sempre le slice a meno che tu non abbia un motivo specifico per una dimensione fissa. append è il tuo strumento quotidiano.

Le condizioni (if/else)

Le condizioni eseguono blocchi di codice diversi a seconda che una condizione sia vera o falsa, con if, else if ed else.

if / else if / else
📘 In Go la condizione non va tra parentesi e le graffe sono obbligatorie, anche per una sola riga.

Lo switch

Lo switch sceglie tra molti casi in modo più leggibile di tanti else if. In Go non serve il break: ogni caso si ferma da solo.

Switch su un valore
💡 A differenza di altri linguaggi, in Go i casi non "cadono" sul successivo: niente break da ricordare. Per il comportamento opposto esiste fallthrough.

I cicli (for)

Go ha un solo costrutto di ciclo: for. Ma è flessibile e copre tutti i casi (incluso il classico while di altri linguaggi).

Le forme del for
📘 Il for ... range è il modo idiomatico di scorrere slice, array e map: restituisce indice e valore di ogni elemento.

Le funzioni

Una funzione raggruppa istruzioni riutilizzabili. In Go i tipi dei parametri e del valore restituito si dichiarano esplicitamente.

Definire e usare funzioni
💡 La possibilità di restituire più valori è una caratteristica amata di Go: si usa moltissimo per restituire un risultato e un eventuale errore.

Le struct

Una struct raggruppa più campi correlati in un unico tipo personalizzato: è il modo di Go per rappresentare "oggetti" come una persona, un prodotto, un punto.

Definire e usare una struct
📘 Le struct sono il cuore di Go: non c'è la parola class, ma combinando struct e metodi si ottiene tutto ciò che serve della programmazione a oggetti.

Le map

Una map associa chiavi a valori (come un dizionario): perfetta per cercare un valore tramite un nome.

Creare e usare una map
💡 La forma valore, esiste := mappa[chiave] è il modo sicuro di controllare se una chiave c'è prima di usarla.

  2. Usa una map per associare 3 nomi alle loro età

🎯 Sfida

Come funziona il compilatore Go

A differenza di Python o JavaScript (interpretati), Go è compilato: il codice sorgente viene tradotto in un eseguibile in codice macchina, velocissimo e senza dipendenze esterne.

I comandi del compilatore
📘 go fmt riformatta il codice secondo lo stile ufficiale: in Go non si discute di stile, è uno solo per tutti. Un grande vantaggio per i team.
