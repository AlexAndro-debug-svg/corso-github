Corso Angular

Angular è una piattaforma e un framework per costruire applicazioni web a pagina singola (SPA) usando HTML e TypeScript. È sviluppato e mantenuto da Google.

Perché scegliere Angular?

  • 🏗️ Framework completo: routing, form, HTTP e testing già inclusi
  • 📐 Struttura solida: ideale per applicazioni grandi ed enterprise
  • 🔷 TypeScript: tipizzazione statica che previene molti errori
  • ⚡ Reattività moderna: con i Signals e il change detection ottimizzato

Esempio di codice

Introduzione ad Angular

Angular è stato creato da Google nel 2016 (la riscrittura completa di AngularJS del 2010). Oggi è uno dei framework frontend più usati nelle grandi aziende.

Concetti fondamentali

  • Componenti: i mattoni dell'interfaccia utente
  • Template: l'HTML con la sintassi Angular
  • Direttive: estendono l'HTML con comportamenti
  • Servizi: logica riutilizzabile e condivisa
  • Dependency Injection: gestione automatica delle dipendenze
  • Moduli / Standalone: organizzazione del codice

Change Detection

Il change detection è il meccanismo con cui Angular aggiorna il DOM quando i dati cambiano.

Come funziona

Angular controlla i componenti e, se rileva cambiamenti nei dati, aggiorna la vista. Di default usa la strategia Default, che controlla tutto l'albero dei componenti.

Strategia OnPush

Esempio di codice

Con OnPush, Angular controlla il componente solo quando:

  • Cambia un riferimento di un @Input
  • Si verifica un evento nel componente
  • Cambia un Signal usato nel template
