Introduzione a Java

Java è un linguaggio di programmazione orientato agli oggetti, creato da Sun Microsystems nel 1995 e oggi mantenuto da Oracle. È uno dei linguaggi più usati al mondo per applicazioni aziendali, app Android, sistemi bancari e software di grandi dimensioni.

Il suo motto è “Write Once, Run Anywhere”: il codice Java viene compilato in bytecode ed eseguito dalla JVM (Java Virtual Machine), perciò gira su qualsiasi sistema operativo senza modifiche. Java è fortemente tipizzato e segue rigorosamente il paradigma a oggetti.

📌 Java non si esegue nel browser: in questo corso gli esempi si leggono e si modificano con la sintassi colorata. Per eseguirli ti serve il JDK installato sul tuo computer (vedi “Installazione”).

Installare Java (Get Started)

Per programmare in Java ti serve il JDK (Java Development Kit), che include il compilatore javac e la JVM. Si scarica da oracle.com o, in versione open source, da adoptium.net. Come editor si usano IntelliJ IDEA, Eclipse o VS Code.

Sintassi e primo programma

Ogni programma Java è organizzato in classi. L'esecuzione parte dal metodo speciale main. Le istruzioni terminano con il punto e virgola ; e i blocchi sono racchiusi tra parentesi graffe { }.

Esempio

Output: System.out.println

Per stampare a schermo si usa System.out.println() (stampa e va a capo) oppure System.out.print() (senza andare a capo).

Esempio

I commenti

I commenti spiegano il codice e non vengono eseguiti: su una riga con //, su più righe con /* ... */.

Esempio

Le variabili

In Java ogni variabile deve essere dichiarata con il suo tipo (tipizzazione statica). La sintassi è tipo nome = valore;. Con final si crea una costante non modificabile.

Esempio

I tipi di dato

Java distingue i tipi primitivi (valori semplici) dai tipi riferimento (oggetti come String, array).

Il casting (conversione di tipo)

Il casting converte un tipo in un altro. Quello widening (es. int → double) è automatico; quello narrowing (es. double → int) va indicato esplicitamente fra parentesi.

Esempio

Gli operatori

Java offre operatori aritmetici (+ - * / %), di confronto (== != > <), logici (&&, ||, !) e di assegnazione (= += -=).

Esempio

Le stringhe

La classe String rappresenta il testo e offre molti metodi: length(), toUpperCase(), substring(), indexOf(). Le stringhe si concatenano con +.

Esempio

La classe Math

La classe Math fornisce funzioni matematiche: Math.sqrt(), Math.pow(), Math.max(), Math.random() e la costante Math.PI.

Esempio

L'input dell'utente

Per leggere dati dalla tastiera si usa la classe Scanner del package java.util.

Esempio

Le condizioni: if, else if, else

Le istruzioni condizionali eseguono codice in base a una condizione booleana.

Esempio

Il costrutto switch

Lo switch confronta una variabile con più valori (case). Usa break per uscire e default per il caso generale.

Esempio

Il ciclo while

Il while ripete un blocco finché la condizione è vera. La variante do...while esegue almeno una volta.

Esempio

Il ciclo for

Il for è ideale quando conosci il numero di ripetizioni. Esiste anche il for-each per scorrere array e collezioni.

Esempio

Gli array

Un array contiene più valori dello stesso tipo, con dimensione fissa. Si accede con l'indice (parte da 0) e si ottiene la lunghezza con .length.

Esempio

I metodi

Un metodo è un blocco di codice riutilizzabile con un nome. void indica che non restituisce nulla; static permette di chiamarlo senza creare un oggetto.

Esempio

Parametri e valore di ritorno

Un metodo può ricevere parametri e restituire un valore con return, indicandone il tipo al posto di void.

Esempio

Programmazione a oggetti (OOP)

Java è un linguaggio profondamente orientato agli oggetti. Tutto ruota attorno a classi (i modelli) e oggetti (le istanze). I pilastri sono incapsulamento, ereditarietà, polimorfismo e astrazione: permettono di scrivere software grande, ordinato e riutilizzabile.

📦 Una classe è lo “stampo”, gli oggetti sono i “biscotti” creati con quello stampo.

Classi e oggetti

Una classe definisce attributi (dati) e metodi (comportamenti). Con new si crea un oggetto.

Esempio

Gli attributi

Gli attributi (o campi) sono le variabili che descrivono lo stato di un oggetto. Ogni oggetto ha la propria copia.

Esempio

I metodi di classe

I metodi definiscono il comportamento degli oggetti e possono usare i loro attributi.

Esempio

I costruttori

Un costruttore è un metodo speciale (stesso nome della classe) chiamato automaticamente con new per inizializzare l'oggetto. La parola chiave this indica l'oggetto corrente.

Esempio

I modificatori d'accesso

I modificatori d'accesso stabiliscono la visibilità di attributi e metodi: public (ovunque), private (solo nella classe), protected (classe e sottoclassi), o di default (stesso package).

ModificatoreVisibilità
publicDa qualsiasi classe
privateSolo dentro la classe
protectedClasse, package e sottoclassi

L'incapsulamento

L'incapsulamento protegge i dati rendendo gli attributi private e fornendo metodi pubblici getter e setter per leggerli e modificarli in modo controllato.

Esempio

L'ereditarietà

Con extends una classe (figlia) eredita attributi e metodi da un'altra (genitore), riutilizzando il codice. super richiama il genitore.

Esempio

Il polimorfismo

Il polimorfismo permette di usare lo stesso metodo con comportamenti diversi a seconda dell'oggetto. Con l'override una sottoclasse ridefinisce un metodo del genitore.

Esempio

Le interfacce

Un'interfaccia definisce un “contratto” di metodi che le classi devono implementare con implements. È il modo di Java per l'astrazione e per ottenere una forma di ereditarietà multipla.

Esempio

ArrayList e collezioni

L'ArrayList è una lista dinamica (a dimensione variabile), parte del Collections Framework. A differenza degli array può crescere e ridursi.

Esempio

La gestione delle eccezioni

Le eccezioni sono errori che si verificano durante l'esecuzione. Si gestiscono con try ... catch (e l'opzionale finally) per evitare che il programma vada in crash.

Esempio
