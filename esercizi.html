🐍 Somma dei numeri pari

Base

Scrivi un programma che calcola la somma di tutti i numeri pari da 1 a 100.

Mostra soluzione 
totale = 0
for n in range(1, 101):
    if n % 2 == 0:
        totale += n
print(totale)   # 2550

🐍 Numero palindromo

Medio

Verifica se una parola inserita dall'utente è palindroma (si legge uguale al contrario).

Mostra soluzione 
parola = input("Inserisci una parola: ")
if parola == parola[::-1]:
    print("È palindroma!")
else:
    print("Non è palindroma.")

🌐 Centrare un box con Flexbox

Medio

Usa CSS Flexbox per centrare un box sia orizzontalmente che verticalmente nella pagina.

Mostra soluzione 
.contenitore {
  display: flex;
  justify-content: center;  /* orizzontale */
  align-items: center;      /* verticale */
  height: 100vh;
}

🗃️ Query SELECT con WHERE

Base

Dalla tabella studenti, estrai nome e cognome di chi ha più di 18 anni.

Mostra soluzione 
SELECT nome, cognome
FROM studenti
WHERE eta > 18;

🗃️ JOIN tra due tabelle

Avanzato

Mostra il nome di ogni studente con la descrizione del corso a cui è iscritto.

Mostra soluzione 
SELECT s.nome, c.descrizione
FROM studenti s
JOIN corsi c ON s.id_corso = c.id;

🧮 Fattoriale ricorsivo

Medio

Scrivi una funzione ricorsiva che calcola il fattoriale di un numero n.

Mostra soluzione 
def fattoriale(n):
    if n <= 1:
        return 1
    return n * fattoriale(n - 1)

print(fattoriale(5))   # 120

🧮 Ricerca del massimo

Base

Trova il valore massimo in una lista di numeri senza usare la funzione max().

Mostra soluzione 
numeri = [3, 7, 2, 9, 4]
massimo = numeri[0]
for n in numeri:
    if n > massimo:
        massimo = n
print(massimo)   # 9
