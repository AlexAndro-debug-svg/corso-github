Introduzione a W3.CSS

W3.CSS è un framework CSS creato da W3Schools che permette di costruire siti web moderni e responsive senza JavaScript e senza dipendenze esterne. Il file pesa meno di 30 KB e include centinaia di classi pronte all'uso.

CaratteristicaDescrizione
Leggero~28 KB, un singolo file CSS
No dipendenzeNon richiede jQuery o altri framework
ResponsiveLayout mobile-first integrato
Cross-browserFunziona su tutti i browser moderni
Dark modeTema scuro incluso

Rispetto a Bootstrap, W3.CSS è più snello e non richiede JavaScript per la maggior parte dei componenti. È ideale per siti statici, portfolio e landing page.

💡 Nota: W3.CSS non sostituisce la conoscenza di CSS puro — è uno strumento per velocizzare il lavoro. Imparare prima il CSS di base rimane fondamentale.

Setup e inclusione

Per usare W3.CSS basta includere il file CSS nel <head> della pagina. Puoi usare il CDN di W3Schools o scaricare il file.

Via CDN (consigliato)

HTML — inclusione CDN
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">
  <title>Il mio sito con W3.CSS</title>
</head>
<body>

<div class="w3-container w3-blue">
  <h1>Benvenuto in W3.CSS!</h1>
</div>

<div class="w3-container">
  <p>Il framework CSS più leggero del mondo.</p>
</div>

</body>
</html>

Struttura base della pagina

Tutte le classi W3.CSS iniziano con il prefisso w3-. La classe fondamentale è w3-container che aggiunge padding orizzontale.

ClasseEffetto
w3-containerPadding left/right 16px
w3-contentContainer centrato max-width 980px
w3-panelPadding top/bottom 0.01em e 16px
w3-mainMargine automatico per sidebar layout
💡 Tip: Usa sempre il tag <meta name="viewport"> per garantire il comportamento responsive su mobile.

Sidebar

Una sidebar laterale è un pattern comune nei dashboard e nei siti di documentazione. Con W3.CSS si costruisce con w3-sidebar e JavaScript minimale.

HTML — sidebar con toggle
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">

<!-- Sidebar -->
<div id="mySidebar" class="w3-sidebar w3-bar-block w3-blue w3-card"
     style="width:200px; display:none">
  <button onclick="chiudiSidebar()"
          class="w3-bar-item w3-button w3-large">&times; Chiudi</button>
  <a href="#" class="w3-bar-item w3-button">🏠 Home</a>
  <a href="#" class="w3-bar-item w3-button">📂 Progetti</a>
  <a href="#" class="w3-bar-item w3-button">📝 Blog</a>
  <a href="#" class="w3-bar-item w3-button">✉️ Contatti</a>
</div>

<!-- Contenuto principale -->
<div id="main">
  <div class="w3-container">
    <button class="w3-button w3-blue w3-margin" onclick="apriSidebar()">
      ☰ Menu
    </button>
    <h2>Contenuto principale</h2>
    <p>Clicca il pulsante Menu per aprire la sidebar.</p>
  </div>
</div>

<script>
function apriSidebar() {
  document.getElementById('mySidebar').style.display = 'block';
  document.getElementById('main').style.marginLeft = '200px';
}
function chiudiSidebar() {
  document.getElementById('mySidebar').style.display = 'none';
  document.getElementById('main').style.marginLeft = '0';
}
</script>

function apriMenu(el) {
  el.nextElementSibling.classList.toggle('w3-show');
}
</script>

</script>

Tabs

Le tab permettono di organizzare contenuti in pannelli selezionabili. W3.CSS le gestisce con w3-bar e JavaScript leggero.

HTML — tabs
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">

<!-- Barra tab -->
<div class="w3-bar w3-teal">
  <button class="w3-bar-item w3-button tablink w3-dark-grey" onclick="apriTab('HTML')">HTML</button>
  <button class="w3-bar-item w3-button tablink" onclick="apriTab('CSS')">CSS</button>
  <button class="w3-bar-item w3-button tablink" onclick="apriTab('JS')">JavaScript</button>
</div>

<!-- Pannelli -->
<div id="HTML" class="w3-container w3-border w3-padding">
  <h2>HTML</h2>
  <p>HTML è il linguaggio di marcatura per le pagine web.</p>
</div>
<div id="CSS" class="w3-container w3-border w3-padding" style="display:none">
  <h2>CSS</h2>
  <p>CSS gestisce lo stile e il layout delle pagine HTML.</p>
</div>
<div id="JS" class="w3-container w3-border w3-padding" style="display:none">
  <h2>JavaScript</h2>
  <p>JavaScript rende le pagine web dinamiche e interattive.</p>
</div>

<script>
function apriTab(nome) {
  document.querySelectorAll('.w3-container').forEach(p => p.style.display = 'none');
  document.querySelectorAll('.tablink').forEach(b => b.classList.remove('w3-dark-grey'));
  document.getElementById(nome).style.display = 'block';
  event.currentTarget.classList.add('w3-dark-grey');
}
</script>

Modal

Un modal è una finestra in sovrapposizione. W3.CSS fornisce la classe w3-modal con un overlay scuro e la classe w3-modal-content per il contenuto.

HTML — modal completo
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">

<!-- Pulsante di apertura -->
<button onclick="document.getElementById('mioModal').style.display='block'"
        class="w3-button w3-blue w3-large">
  Apri Modal
</button>

<!-- Modal -->
<div id="mioModal" class="w3-modal" onclick="this.style.display='none'">
  <div class="w3-modal-content w3-card w3-animate-top" onclick="event.stopPropagation()">
    <header class="w3-container w3-teal">
      <span onclick="document.getElementById('mioModal').style.display='none'"
            class="w3-button w3-display-topright">&times;</span>
      <h2>Titolo Modal</h2>
    </header>
    <div class="w3-container w3-padding-32">
      <p>Questo è il contenuto del modal. Puoi inserire testo, form, immagini, ecc.</p>
      <button onclick="document.getElementById('mioModal').style.display='none'"
              class="w3-button w3-red">Chiudi</button>
      <button class="w3-button w3-green">Conferma</button>
    </div>
  </div>
</div>
💡 Tip: w3-animate-top fa entrare il modal dall'alto con un'animazione fluida. Prova anche w3-animate-zoom per un effetto zoom-in.

Dark Mode

W3.CSS supporta il tema scuro includendo il file w3-theme-black.css oppure usando le classi di colore scuro già incluse.

HTML — dark mode con toggle
<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">

<style>
  body { background: #fff; color: #333; transition: background 0.3s, color 0.3s; }
  body.dark { background: #1a1a1a; color: #e0e0e0; }
  body.dark .w3-card { background: #2d2d2d; color: #e0e0e0; }
</style>

<div class="w3-bar w3-black">
  <span class="w3-bar-item w3-large">🌙 Dark Mode Demo</span>
  <button onclick="toggleDark()" class="w3-button w3-right w3-yellow">
    ☀️/🌙 Toggle
  </button>
</div>

<div class="w3-container w3-padding-32">
  <div class="w3-card w3-padding-16 w3-margin">
    <h2>Titolo</h2>
    <p>Questo contenuto cambia aspetto con la dark mode.</p>
    <button class="w3-button w3-blue">Pulsante</button>
  </div>
</div>

<script>
function toggleDark() {
  document.body.classList.toggle('dark');
}
</script>
💡 W3Schools offre anche temi precostruiti come w3-theme-dark scaricabili dal sito ufficiale, che cambiano automaticamente tutti i colori del tema.

🏆 Esercizio 1 — Pagina portfolio

Crea una landing page con: navbar nera con 4 voci + pulsante "Contatti" a destra, sezione hero con sfondo teal e testo centrato, 3 card affiancate con icona, titolo e descrizione, footer scuro. Usa solo classi W3.CSS.
