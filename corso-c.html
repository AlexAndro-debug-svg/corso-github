E ora?

Introduzione al linguaggio C

C è un linguaggio di programmazione creato da Dennis Ritchie nei laboratori Bell nel 1972. È uno dei linguaggi più influenti della storia: da lui derivano C++, C#, Java, JavaScript e molti altri. È usato per sistemi operativi (Linux, Windows), driver, sistemi embedded, microcontrollori e tutto ciò che richiede massima velocità ed efficienza.

C è un linguaggio compilato, procedurale e di “basso livello” rispetto ad altri: dà al programmatore il controllo diretto della memoria. È più severo e tecnico, ma capirlo ti fa comprendere davvero come funziona un computer.

📌 C non si esegue nel browser: serve un compilatore (es. GCC). In questo corso gli esempi si leggono e si modificano con la sintassi colorata.

Installare C (Get Started)

Per compilare codice C ti serve un compilatore. Il più diffuso è GCC (su Linux/macOS spesso già presente, su Windows tramite MinGW). Come editor si usano VS Code, Code::Blocks o Dev-C++.

Terminale

Sintassi e primo programma

Ogni programma C parte dalla funzione main(). La direttiva #include importa librerie (qui stdio.h per l'input/output). Le istruzioni terminano con ; e i blocchi sono tra { }.

Esempio

Output: la funzione printf

printf() stampa testo a schermo. Usa gli specificatori di formato per inserire valori: %d (interi), %f (decimali), %c (caratteri), %s (stringhe). \n manda a capo.

Gli array

Un array contiene più valori dello stesso tipo, con dimensione fissa. Si accede con l'indice (da 0).

Esempio

Le stringhe

In C una stringa è un array di caratteri che termina con il carattere nullo '\0'. La libreria string.h offre funzioni come strlen(), strcpy(), strcat().

Esempio

Le funzioni

Una funzione raggruppa codice riutilizzabile, con un tipo di ritorno, un nome e dei parametri. Si dichiara prima del main (o con un prototipo).

Esempio

I puntatori

I puntatori sono la caratteristica più potente (e temuta) di C: una variabile che contiene l'indirizzo di memoria di un'altra variabile. Si dichiarano con * e si ottiene l'indirizzo con &.

Esempio

Le struct

Una struct raggruppa più variabili (anche di tipo diverso) in un unico tipo personalizzato: è l'antenato degli oggetti.

Esempio

La gestione della memoria

In C il programmatore può allocare memoria dinamicamente con malloc() e liberarla con free() (libreria stdlib.h). È potente ma rischioso: dimenticare free() causa memory leak.
