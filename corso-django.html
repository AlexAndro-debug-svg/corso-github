E ora?

Cos'è Django

Django è un framework web Python ad alto livello che incoraggia lo sviluppo rapido e un design pulito e pragmatico. È stato creato nel 2003 e oggi è usato da alcuni dei siti più trafficati al mondo, come Instagram, Pinterest e Spotify.

Perché scegliere Django

  • ⚡ Veloce: porta le applicazioni dall'idea al completamento rapidamente
  • 🛡️ Sicuro: aiuta a evitare i comuni errori di sicurezza
  • 📈 Scalabile: regge siti con milioni di utenti
  • 🔋 Batterie incluse: admin, ORM, autenticazione e form già pronti

Introduzione a Django

Django segue il pattern architetturale MTV (Model-Template-View), una variante del classico MVC.

Il pattern MTV

  • Model: gestisce i dati e la logica del database
  • Template: gestisce la presentazione (l'HTML mostrato all'utente)
  • View: gestisce la logica che collega Model e Template

"Batterie incluse"

Django arriva con molte funzionalità pronte all'uso:

  • Pannello di amministrazione automatico
  • ORM (Object-Relational Mapping) per il database
  • Sistema di autenticazione utenti
  • Sistema di form e validazione
  • Internazionalizzazione e localizzazione

Prerequisiti

  • Conoscenza base di Python
  • Concetti base di HTML e CSS
  • Nozioni base di database

Creare un Account AWS

AWS RDS (Relational Database Service) permette di ospitare PostgreSQL nel cloud.

Passi per creare l'account

  • Vai su aws.amazon.com e clicca "Crea un account AWS"
  • Inserisci email, password e nome account
  • Aggiungi un metodo di pagamento (il Free Tier è gratuito per 12 mesi)
  • Verifica l'identità e scegli il piano "Basic"

Free Tier per RDS

  • 750 ore/mese di istanza db.t3.micro
  • 20 GB di storage SSD
  • 20 GB di backup automatico
Esempio di codice

Creare un Database su RDS

Dalla console AWS si crea un'istanza PostgreSQL gestita.

Passaggi principali

  • Cerca RDS nella console e clicca "Create database"
  • Engine: PostgreSQL, Template: Free tier
  • Imposta identificatore, username e password
  • Instance class: db.t3.micro, Storage: 20 GB
  • Public access: Yes (per lo sviluppo)
Esempio di codice
