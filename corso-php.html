Introduzione a PHP

PHP (PHP: Hypertext Preprocessor) è un linguaggio di scripting lato server creato nel 1994, pensato appositamente per lo sviluppo web. Gira sul server: il codice viene eseguito lì e al browser arriva solo il risultato in HTML. È il motore di WordPress, Wikipedia, Facebook (alle origini) e di gran parte del web.

Con PHP puoi creare siti dinamici, gestire i dati inviati dai form, dialogare con database come MySQL, gestire login, sessioni e cookie. È facile da imparare e si integra direttamente nell'HTML.

📌 PHP non si esegue nel browser: ha bisogno di un server con interprete PHP (es. XAMPP in locale, oppure un hosting). In questo corso gli esempi si leggono e si modificano con la sintassi colorata.
Esempio

Installare PHP (Get Started)

Il modo più semplice per provare PHP sul tuo computer è installare XAMPP (o MAMP/WAMP): un pacchetto che include il server Apache, PHP e MySQL. I file PHP (estensione .php) vanno messi nella cartella htdocs e si aprono dal browser all'indirizzo http://localhost/file.php.

💡 In alternativa, ogni hosting web (Aruba, SiteGround, Altervista...) supporta PHP: carichi i file via FTP e funzionano subito.

La sintassi di PHP

Il codice PHP si scrive tra i tag <?php e ?> e può essere inserito ovunque dentro una pagina HTML. Ogni istruzione termina con il punto e virgola ;.

Esempio

Stampare: echo e print

echo e print mostrano contenuti nella pagina. echo è leggermente più veloce e può stampare più valori separati da virgola.

Esempio

I commenti

PHP supporta commenti su una riga con // o #, e su più righe con /* ... */.

Esempio

Le variabili

In PHP le variabili iniziano sempre con il simbolo dollaro $. Non serve dichiarare il tipo: viene dedotto dal valore. Sono case-sensitive.

Esempio

I tipi di dato

I tipi principali in PHP sono: string (testo), int (interi), float (decimali), bool (true/false), array, object e NULL. La funzione gettype() mostra il tipo.

Esempio

Le stringhe

Le stringhe si concatenano con il punto . e offrono molte funzioni: strlen(), strtoupper(), str_replace(), substr().

Esempio

Numeri e funzioni matematiche

PHP gestisce interi e decimali e offre funzioni matematiche come abs(), round(), sqrt(), pow(), rand().

Esempio

Le costanti

Una costante è un valore che non cambia. Si definisce con define() o con const e, per convenzione, si scrive in MAIUSCOLO. Non usa il $.

Esempio

Gli operatori

PHP offre operatori aritmetici (+ - * / %), di confronto (== === != > <) e logici (&& || !). Attenzione: == confronta i valori, === confronta valore E tipo.

Esempio

Le condizioni: if, elseif, else

Eseguono codice diverso a seconda di una condizione. Esiste anche l'operatore ternario ?: per le scelte rapide.

Esempio

Il costrutto switch

Lo switch confronta una variabile con più casi, alternativa più leggibile a tanti elseif.

Esempio

Il ciclo while

Il while ripete un blocco finché la condizione è vera.

Esempio

Il ciclo for

Il for è adatto quando conosci il numero di ripetizioni.

Esempio

Le funzioni

Una funzione raggruppa codice riutilizzabile. Si definisce con function, può avere parametri (anche con valore di default) e restituire un valore con return.

Esempio

Gli array

Un array contiene più valori. In quello indicizzato si accede con un numero (da 0). Si crea con array() o [].

Esempio

Gli array associativi

Negli array associativi ogni valore ha una chiave testuale, come un dizionario. Perfetti per rappresentare dati strutturati.

Esempio

Il ciclo foreach

Il foreach scorre tutti gli elementi di un array, anche associativo (con chiave e valore).

Esempio

Le date

La funzione date() formatta la data e l'ora correnti del server secondo uno schema di caratteri (Y = anno, m = mese, d = giorno, H:i:s = ora).

Esempio

Include e require

include e require inseriscono il contenuto di un altro file PHP: utilissimi per riusare header, footer e menu su tutte le pagine. La differenza: se il file manca, require blocca lo script, include dà solo un avviso.

Esempio

Gestire i form: $_POST

PHP riceve i dati inviati da un form HTML tramite le superglobali $_POST (metodo POST) e $_GET (metodo GET). Il valore di name di ogni campo diventa la chiave.

Esempio

I parametri nell'URL: $_GET

Con il metodo GET i dati viaggiano nell'URL (es. pagina.php?id=5) e si leggono da $_GET. È adatto a ricerche e link, ma non a dati sensibili (sono visibili nell'URL).

Esempio

La validazione dei dati

Non fidarti mai dei dati dell'utente! Vanno sempre controllati e ripuliti per sicurezza. Funzioni utili: empty(), isset(), htmlspecialchars() (contro gli attacchi XSS) e filter_var() per validare email.

Esempio

Sessioni e cookie

Le sessioni conservano dati lato server tra una pagina e l'altra (es. per il login). I cookie salvano dati nel browser dell'utente. Una sessione si avvia con session_start() e si usa l'array $_SESSION.

Esempio

OOP: classi e oggetti

PHP supporta la programmazione a oggetti. Una classe definisce proprietà e metodi; con new si crea un oggetto. Il costruttore __construct() inizializza l'oggetto; $this indica l'oggetto corrente.

Esempio

L'ereditarietà

Con extends una classe eredita proprietà e metodi da un'altra. I modificatori public, private e protected controllano la visibilità (incapsulamento).

Esempio

PHP e MySQL

La combinazione PHP + MySQL è alla base dei siti dinamici: PHP si connette al database, esegue query e mostra i risultati. Il modo moderno e sicuro è PDO, che protegge dalle SQL injection con i prepared statement.

Esempio
🔒 Usa sempre i prepared statement (i ? al posto dei valori): inserire variabili direttamente nella query apre la porta agli attacchi di SQL injection.

Complimenti! 🎉 Hai completato il corso di PHP

Hai imparato la sintassi di PHP, le variabili e i tipi, gli array, le funzioni, la gestione dei form con $_POST e $_GET, le sessioni, la programmazione a oggetti e la connessione a MySQL: tutto ciò che serve per costruire siti web dinamici.
